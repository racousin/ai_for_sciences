# AI for Sciences - Winter School Practical Works

This repository contains the practical work materials for the [Joint Winter School on AI for Sciences](https://event.ntu.edu.sg/Joint-Winter-School-AI-for-Sciences).

## Organization

The practical sessions are organized as Jupyter notebooks that can be executed directly in Google Colab. A Gmail account is required to access Google Colab.

- **tp2.ipynb**: Text representation, tokenizers, and byte-pair encoding (BPE)  GPT architecture, one-shot learning with Hugging Face, and fine-tuning
